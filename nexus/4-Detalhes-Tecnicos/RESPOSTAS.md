# 4. Detalhes Técnicos da Solução

### Qual é o tipo de solução criada?
**R:** A solução desenvolvida é uma **Plataforma de Inteligência Territorial e Transição Energética (Web Application)**. Trata-se de um dashboard interativo (SaaS) projetado para centralizar dados geoespaciais, acervos científicos e assistência por Inteligência Artificial para o ecossistema de descarbonização.

### Qual (ou quais) linguagem(ns) de programação (e/ou framework) utilizada(s) no desenvolvimento?
**R:** A plataforma foi desenvolvida utilizando tecnologias de ponta para garantir performance e escalabilidade:
- **Framework Principal:** Next.js 15 (App Router)
- **Linguagem:** TypeScript (Garantindo tipagem forte e segurança de código)
- **Estilização:** Tailwind CSS (Design System institucional personalizado)
- **Componentes:** shadcn/ui (Interface premium e acessível)
- **Mapas:** React-Leaflet com integração de camadas ESRI e CartoDB
- **Ícones:** Lucide React
- **Desenvolvimento:** AI-Assisted Professional Coding (utilizando Antigravity para alta fidelidade e lógica complexa)

### Qual é o manual da solução criada?
**R:** 
1. **Acesso:** A solução está disponível publicamente via URL: [https://nexus-alpha-liard-60.vercel.app](https://nexus-alpha-liard-60.vercel.app).
2. **Navegação:** O usuário utiliza a barra lateral (Sidebar) para alternar entre os módulos:
   - **Mapa (IpnMap):** Visualização de pontos estratégicos de energia e recursos naturais na América Latina.
   - **Base de Dados:** Pesquisa semântica em mais de 20 fontes oficiais (IBGE, EPE, ANP, etc.).
   - **Copilot:** Assistente de IA para suporte em pesquisas e análises de dados.
   - **Simulador/Matching/Marketplace:** Ferramentas para conexão entre produtores e investidores.
3. **Interação:** Todos os elementos são interativos. Os filtros no Mapa permitem mudar a visualização do terreno e as buscas na Base de Dados redirecionam diretamente para as fontes oficiais.
